Understanding Objects: The Person Example

This page demonstrates how to create a 'Person' object using a special JavaScript function called a 'constructor'.

Open your browser's developer console (usually by pressing F12) to see the details of the created people!

What's Happening Behind the Scenes?

In the JavaScript code, we're defining a blueprint for a 'Person'. Then, we're using that blueprint to create two actual people: Alice and Bob. Their details will be printed in your console.
